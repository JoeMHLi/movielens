HarvardX: PH125.9x: Data Science - Capstone - Movielens Recommendation System Project

This repository is created to file and share one of the two capstone projects of the HarvardX Professional Certificate of Data Science.

This project is to develop an algorithm for a movie recommendation system using the MovieLens 10M dataset such that it is capable to predict ratings with a root mean square error of less than 0.86490.
